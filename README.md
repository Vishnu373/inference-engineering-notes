## 🎯 What is Inference Engineering?

Inference is when an AI model responds to users in production. It's different from training or research — it's the model doing its job at scale. This repository covers everything that goes into making inference work reliably and efficiently.

## 📖 Topics Covered

- **3 Core Layers**: Runtime, Infrastructure, and Tooling
- **Performance Techniques**: Batching, caching, quantization, speculative decoding, parallelism, and disaggregation  
- **Framework Stack**: CUDA, PyTorch, vLLM, SGLang, and TensorRT-LM
- **Deployment Patterns**: Multi-cloud, edge inference, and Inference-as-a-Service
